Card Library | Using CSS Grid at the Component Level

Below are examples of a card component created using CSS Grid. Each card is it's own self contained CSS grid, starting out with two Grid Columns defined.

All cards are created using the same markup. The different layouts are achieved through line-based positioning of the card's direct child elements. The look of the card changes by adding a css class on the parent .card div

Cards can be used by themselves, or placed within page or module level grids.
